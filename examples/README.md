## 使用方式

复制示例中的用户问题，直接发给 AI（Claude、Cursor、OpenClaw 等），即可完成对应操作。

每个示例包含：
- **用户提问** — 你可以对 AI 说的话
- **调用工具** — AI 实际执行的 MCP 工具
- **返回结果** — 典型返回数据（格式参考）
- **注意事项** — 常见问题和最佳实践
